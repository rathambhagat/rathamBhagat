> 💡 *To enable the snake animation, add the [Platane/snk](https://github.com/Platane/snk) GitHub Action to this repo. See setup instructions below.*

<details>
<summary>🔧 Snake Action Setup (Click to expand)</summary>

Create `.github/workflows/snake.yml` in this repository:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: rathambhagat
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

</details>
